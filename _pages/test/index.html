How to Check if Gowun Batang is Working:

  1. Visual Comparison: Compare the first section (Gowun Batang) with the last section (Default Serif). They should look different if the Google Font is loading.
  2. Korean Characters: The Korean text should display properly if Gowun Batang is loaded.
  3. Developer Tools:
    • Right-click on any Gowun Batang text and select "Inspect"
    • In the Computed tab, look for "font-family"
    • It should show "Gowun Batang" as the computed font
  4. Network Tab:
    • Open Developer Tools → Network tab
    • Refresh the page
    • Filter by "Font" - you should see font files being downloaded from Google Fonts
